View GOV.UK One Login user journeys

Sign in journeys ('authentication')

Create a GOV.UK One Login or sign in with an existing one

A user starts on a service page, then can access that service by creating a GOV.UK One Login for the first time, or signing in with existing one.

Identity checking journeys

Both these journeys include a user signing in using an existing GOV.UK One Login.

Passport + knowledge-based verification (KBV) questions in a web browser

User completes the whole journey in their browser. They have a passport but do not use the app, so they answer security questions instead.

NFC-chipped passport + the GOV.UK ID Check app

User starts the journey in their browser. They have a passport with an NFC chip in it and choose to prove their identity using the GOV.UK ID Check app.

The signed in experience

Change GOV.UK One Login credentials

Users can navigate from the service they’re in to their GOV.UK One Login to change their sign in details.
